# ESLint shareable config for canX

## Install

```
$ npm install -D -E eslint prettier @canx/eslint-config
```

## Usage

Add some ESLint config to your `package.json`:

```json
{
	"name": "my-awesome-project",
	"eslintConfig": {
		"extends": "@canx/eslint-config"
	}
}
```

Or to `.eslintrc.json`:

```json
{
	"extends": "@canx/eslint-config"
}
```
